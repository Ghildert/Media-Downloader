# 🎬 YT-DLP Tauri Downloader
A lightweight, desktop media downloader built with Tauri and powered by yt-dlp. 
Download videos, audio, and playlists with a simple UI.

## 🚀 Features
- 📥 Download videos
- 🎵 Extract audio (MP3, M4A, etc.)
- 📺 Supports playlists and channels
- ⚡ Fast and lightweight (thanks to Tauri)
